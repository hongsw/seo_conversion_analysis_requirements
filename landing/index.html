SEO Content Conversion Analytics

콘텐츠 SEO 및 구매 전환율 분석 오픈소스 프로젝트

주요 기능

  • 콘텐츠 트래픽 분석 (페이지별 유입, 체류시간, 유입경로)
  • 전환율 분석 (콘텐츠 → 구매 페이지 전환)
  • SEO 성과 추적 (검색어별 성과, 순위 변화 모니터링)
  • 실시간 대시보드 및 데이터 시각화
  • 자동화된 성과 리포트 생성

기술 스택

Python Django React Chart.js PostgreSQL Docker
